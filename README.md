# Task-05: Web Scraping

## Overview

This project extracts product details like **name, price, and rating** from a website and saves them into a **CSV file** using Python.

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Pandas

## Website Used

http://books.toscrape.com/

## How to Run

1. Install libraries:

   ```bash
   pip install requests beautifulsoup4 pandas
   ```
2. Run the program:

   ```bash
   python scraper.py
   ```

## Output

* `products.csv` file containing scraped data
